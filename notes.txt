------- Basis -------

------- Git Reset -------  
git reset --soft HEAD~1 = löscht letze änderung local aus git 
git reset --mixed HEAD~1
git reset --hard HEAD~1 = löscht die letzte änderung von git und local 

------- Git Revert ------- 
git revert commitID(commitID durch echte id tauschen) = mit revert löschen von github
